Rehearsal Room 🎵

Rehearsal Room is a modern worship team management platform built to help churches and music ministries organize rehearsals, manage songs, track attendance, and improve team communication.

🚀 Features
🎶 Song library management
👥 Choir and worship team member management
📅 Rehearsal scheduling
✅ Attendance tracking
💡 Song suggestion system
🛡️ Admin approval workflow
🔔 Notifications dashboard
📱 Responsive modern UI
🛠️ Tech Stack
Frontend
React
Vite
Tailwind CSS
Backend
ASP.NET Core Web API
Entity Framework Core
Database
SQLite
🔥 Current Functionality
Members can submit song suggestions
Suggestions are stored in the database
Admins can approve or reject songs
Status updates sync live between frontend and backend
Data persists using SQLite
📦 Getting Started
Frontend
cd rehearsalroom-ui
npm install
npm run dev
Backend
cd RehearsalRoomAPI
dotnet run
🌍 API Access

Swagger API documentation:

http://localhost:5281/swagger
🎯 Vision

Rehearsal Room aims to become an all-in-one platform for worship teams and churches by simplifying rehearsal preparation, communication, and collaboration.
